# UI Component Library (Draft)

## Purpose
Define reusable UI components, their variants, and usage guidelines.

## Scope
- Marketing pages
- Authenticated library/player views

## Core Components (Outline)
- Buttons (primary/secondary/ghost)
- Cards (chapter cards)
- Audio player controls
- Navigation and footer
- Forms (email login)

## Component Requirements
- Accessibility baseline (keyboard + focus)
- Audio-first ergonomics
- Responsive behavior
