# Creator OS User Guide

## Purpose
Creator OS is the internal toolset for authoring narrative canon, generating audio scenes, and managing proposals.

## Core Workflows
### 1. Create Narrative Events
1. Draft events in the Narrative Editor.
2. Link dependencies and knowledge updates.
3. Submit as a proposal for validation.

### 2. Review Canon Gate
1. Run validation pipeline.
2. Resolve errors (continuity, dependency DAG).
3. Approve for canonization.

### 3. Generate Audio Scenes
1. Select canon events for export.
2. Produce Audio Scene Object.
3. Validate audio schema and safety thresholds.
4. Export to Audio Engine for rendering.

## Common Actions
- View promise lifecycle status in the Canon Dashboard.
- Inspect knowledge state diffs between drafts and canon.
- Track proposal audit logs.
