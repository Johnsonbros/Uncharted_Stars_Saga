# UI Design System (Draft)

## Purpose
Define the visual language for the Listener Platform UI, including typography, color palette, spacing, and layout rules.

## Scope
- Public marketing pages
- Authenticated library/player views

## Tokens (Outline)
- **Typography**: font families, sizes, weights, line heights
- **Color**: primary/secondary/neutral, semantic states
- **Spacing**: spacing scale and grid
- **Elevation**: shadows and layering

## Usage Guidelines (Outline)
- CTA hierarchy and button prominence
- Audio-first layout prioritization
- Minimum contrast targets

## Open Questions
- Final font selection
- Brand tone and visual references

## Update Triggers
- Visual redesigns
- Brand or marketing updates
